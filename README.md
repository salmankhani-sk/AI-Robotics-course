## ✅ Week 1 Completed - Foundation of Python

Congratulations! You've completed **Week 1**, covering the **Foundation of Python**. Here's what you've learned:

### 📖 Topics Covered in Week 1:

#### 1. **Variables & Data Types** 📝
   - Understanding variables as containers for data
   - Working with different data types (strings, integers, floats, booleans)
   - Type conversion and casting

#### 2. **Python Operators** ⚙️
   - **Arithmetic Operators**: `+`, `-`, `*`, `/`, `//`, `%`, `**`
   - **Comparison Operators**: `==`, `!=`, `>`, `<`, `>=`, `<=`
   - **Logical Operators**: `and`, `or`, `not`
   - Understanding operator precedence

#### 3. **Data Structures** 📦
   - **Lists**: Ordered, mutable collections (can be modified)
   - **Tuples**: Ordered, immutable collections (cannot be modified)
   - **Sets**: Unordered, unique items only
   - **Dictionaries**: Key-value pairs for storing related data
   - Nested data structures and indexing

#### 4. **Control Flow** 🔄
   - **if/elif/else statements**: Making decisions in your code
   - **for loops**: Looping a known number of times
   - **while loops**: Conditional looping
   - Loop control with `break` and `continue`

#### 5. **Functions** 🎯
   - Defining functions with `def` keyword
   - Function parameters and arguments
   - Return statements and return values
   - Default parameters
   - Reusable code blocks

#### 6. **Exception Handling** ⚠️
   - Understanding exceptions and errors
   - **try-except blocks**: Gracefully handling errors
   - **Specific exceptions**: `ZeroDivisionError`, `ValueError`, `TypeError`, `IndexError`
   - **finally blocks**: Code that always executes
   - Preventing program crashes

#### 7. **Lambda Functions** ⚡
   - Anonymous functions using `lambda` keyword
   - Creating quick, single-expression functions
   - Use cases for lambda functions
